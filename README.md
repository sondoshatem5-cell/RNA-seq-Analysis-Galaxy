# RNA-seq Analysis Using Galaxy

## Overview

This repository contains an academic team project completed as part of the Bioinformatics course at Misr University for Science and Technology (MUST).

The project demonstrates a complete RNA-seq analysis workflow using the Galaxy platform, starting from raw sequencing data and ending with differential gene expression and functional enrichment analysis.

---

## Project Type

Academic Team Project

---

## My Contribution

I was responsible for the read alignment and quality assessment stages of the RNA-seq analysis workflow.

My contributions included:

- Performed read alignment using HISAT2 against the reference genome.
- Processed alignment files using SAMtools, including BAM file sorting and indexing.
- Generated and interpreted quality control reports using MultiQC to assess sequencing and alignment quality.
- Verified alignment quality before proceeding to downstream differential gene expression analysis.

Through this work, I gained hands-on experience with RNA-seq data processing, alignment workflows, quality assessment, and bioinformatics analysis using the Galaxy platform.

## Workflow

1. Data collection from NCBI BioProject and SRA.
2. Quality control of raw sequencing data.
3. Read alignment using HISAT2.
4. BAM file processing using SAMtools.
5. Quality assessment using MultiQC.
6. Gene quantification using featureCounts.
7. Differential gene expression analysis using DESeq2.
8. Functional enrichment analysis and biological interpretation.
